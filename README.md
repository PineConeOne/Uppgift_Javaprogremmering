## Del 1

### Objekt

Ett objekt är en instans av en klass som har specifika egenskaper och beteenden.

### Konstruktor

En speciell metod som skapar ett objekt från klassen som konstruktorn tillhör och anger de inparametrar(attribut) som objektet ska ha.

### Attribut

Egenskaper som ett objekt kan ha.

### Versionshantering

Det innebär att hantera olika versioner av källkodsfiler, program eller hemsidor så att man kan se deras historik. Flera personer kan även jobba på olika ställen i samma program samtidigt.

### TDD

TDD är ett sätt att skriva test på, man skriver testet först med syfte att det ska misslyckas(Red). Efter det skrivs koden som gör att testet lyckas(Green). Sist så effektiviseras koden utan att ändra funktionaliteten hos koden så att testerna fortfarande blir lyckade.
